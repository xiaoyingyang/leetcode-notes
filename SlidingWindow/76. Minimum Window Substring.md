## 题目描述
Given two strings s and t of lengths m and n respectively, return the minimum window substring of s such that every character in t (including duplicates) is included in the window. If there is no such substring, return the empty string "".

The testcases will be generated such that the answer is unique.

---

## follow up :
Q: How would your logic change if you were asked to find the smallest window containing at least one of each distinct character in t instead of all occurrences?
A: In the follow-up, I no longer need exact frequencies from t.
I only care about whether each distinct character in t appears at least once in the window.
So I use a set for t, and increment formed when a character’s count changes from 0 to 1, and decrement it when the count changes from 1 to 0.

## 我的代码 - for following up

```java
class Solution {
    public String minWindow(String s, String t) {
        int m = s.length();
        int n = t.length();
        if (m == 0|| n == 0) return "";
        
        Set<Character> need = new HashSet();
        for (char c : t.toCharArray()) {
            need.add(c);
        }
        Map<Character, Integer> window = new HashMap<>();

        int required = need.size();
        int formed = 0;

        int left = 0;
        int minLen = Integer.MAX_VALUE;
        int bestL = 0;
        
        for (int right = 0; right < m; right++){
            char c = s.charAt(right);
            if (need.contains(c)) {
                window.put(c, window.getOrDefault(c, 0) + 1);
                if (window.get(c) == 1) formed++;
            }

            while (formed == required) {
                int len = right - left + 1;
                if (len < minLen) {
                    minLen = len;
                    bestL = left;
                }
                //we shrink the window
                char leftChar = s.charAt(left);
                if (need.contains(leftChar)) {
                    window.put(leftChar, window.get(leftChar)-1);
                    if (window.get(leftChar) == 0) formed--;
                }
                left++;
            }
        }
        return minLen == Integer.MAX_VALUE ? "" : s.substring(bestL, bestL+minLen);
    }
}
```
